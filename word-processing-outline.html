Word Processing & Windows

Master Microsoft Word, Windows basics, and document formatting skills

Course Overview

This course introduces students to essential word processing and Windows operating system skills. Participants will learn to create, edit, format, and manage professional documents while becoming confident in Windows navigation.

Objectives

  • Understand the Windows operating environment
  • Create and manage documents with Microsoft Word
  • Format text, tables, and layouts
  • Use editing and proofing tools effectively
  • Prepare documents for printing and sharing

Course Outline

  • Module 1: Introduction to Windows Environment
  • Module 2: Getting Started with Word Processing
  • Module 3: Formatting and Styles
  • Module 4: Working with Tables and Graphics
  • Module 5: Proofing and Reviewing Documents
  • Module 6: Printing and File Management

Duration

4 Weeks (12 sessions)

Who Can Apply

Beginners, office workers, students, and professionals seeking essential computer skills.

What You'll Gain

Practical word processing skills, confidence in document creation, and improved efficiency in office tasks.
